9) Database Schema (PostgreSQL)
========================
Create sql/postgres_ddl.sql AND docs/database.md describing:
Tables (minimum):
1) stock_master
   - ticker text NOT NULL
   - market text NOT NULL
   - name text NOT NULL
   - listing_date date NULL
   - status text NOT NULL
   - last_seen_date date NOT NULL
   - source text NOT NULL
   - updated_at timestamptz NOT NULL
   - PRIMARY KEY (ticker, market)
2) stock_master_snapshot
   - snapshot_id uuid PK
   - as_of_date date NOT NULL
   - source text NOT NULL
   - fetched_at timestamptz NOT NULL
   - record_count int NOT NULL
3) stock_master_snapshot_items (optional but recommended for auditability)
   - snapshot_id uuid FK
   - ticker, market, name, listing_date, status
   - UNIQUE(snapshot_id, ticker, market)
4) daily_ohlcv
   - trade_date date NOT NULL
   - ticker text NOT NULL
   - market text NOT NULL
   - open/high/low/close bigint NOT NULL
   - volume bigint NOT NULL
   - source text NOT NULL
   - fetched_at timestamptz NOT NULL
   - PRIMARY KEY (trade_date, ticker, market)
   - indexes: (ticker, market, trade_date desc)
5) ingestion_runs
   - run_id uuid PK
   - run_type text (universe_sync|daily_backfill|validate)
   - started_at, ended_at, status, params jsonb, counts jsonb, error_summary text

Upsert plan:
- daily_ohlcv: INSERT ... ON CONFLICT (trade_date, ticker, market) DO UPDATE SET ... (or DO NOTHING) — choose and justify
- stock_master: UPSERT by (ticker, market)
